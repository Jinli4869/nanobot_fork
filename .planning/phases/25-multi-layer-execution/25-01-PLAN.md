Phase 24 inputs:
```python
@dataclass(frozen=True)
class StateDescriptor:
    kind: str
    value: str
    negated: bool = False

@dataclass(frozen=True)
class ParameterSlot:
    name: str
    type: str
    description: str

@dataclass(frozen=True)
class ShortcutSkill:
    skill_id: str
    name: str
    description: str
    app: str
    platform: str
    steps: tuple[SkillStep, ...] = ()
    parameter_slots: tuple[ParameterSlot, ...] = ()
    preconditions: tuple[StateDescriptor, ...] = ()
    postconditions: tuple[StateDescriptor, ...] = ()
```

Grounding contract to reuse directly:
```python
@runtime_checkable
class GrounderProtocol(Protocol):
    async def ground(self, target: str, context: GroundingContext) -> GroundingResult: ...
```

Action normalization boundary:
```python
def parse_action(payload: dict[str, typing.Any]) -> Action: ...
```

Concrete module placement decision for this phase:
```text
- create `opengui/skills/multi_layer_executor.py`
- do not extend `opengui/skills/executor.py`
- keep the legacy executor untouched because it encodes fail-open validation and template fallback semantics that conflict with Phase 25
```

Concrete Phase 25 shortcut result/report target:
```python
@dataclass(frozen=True)
class ContractViolationReport:
    skill_id: str
    step_index: int
    failed_condition: StateDescriptor
    boundary: Literal["pre", "post"]
    is_violation: Literal[True] = True

@dataclass(frozen=True)
class ShortcutStepResult:
    step_index: int
    action: Action
    backend_result: str
    grounding: GroundingResult | None
    screenshot_path: str

@dataclass(frozen=True)
class ShortcutExecutionSuccess:
    skill_id: str
    step_results: tuple[ShortcutStepResult, ...]
    is_violation: Literal[False] = False
```
