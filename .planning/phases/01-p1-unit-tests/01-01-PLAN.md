    - MemoryStore.remove() deletes entry and persists removal
    - MemoryStore.list_all() returns all stored entries
    - MemoryRetriever with _FakeEmbedder returns ranked results from FAISS+BM25 hybrid search
    - MemoryRetriever with alpha=0.0 (BM25-only) ranks by term overlap
    - MemoryRetriever with alpha=1.0 (FAISS-only) ranks by embedding similarity
    - MemoryRetriever.format_context() returns a non-empty formatted string
  </behavior>
  <action>
Create `tests/test_opengui_p1_memory.py` with unit tests for the memory module. Follow the established P0 test patterns:

1. Define a `_FakeEmbedder` class at top of file (from research Pattern 3): returns deterministic float32 unit vectors via `async def embed(self, texts: list[str]) -> np.ndarray`. Each text gets a unique unit vector so FAISS ranking is deterministic.

2. Write sync tests (plain `def test_*()`) for:
   - `test_memory_entry_round_trip`: Create MemoryEntry, call to_dict(), call from_dict() on result, assert equality of all fields
   - `test_memory_store_add_and_reload`: Create MemoryStore(tmp_path), add entry, create new MemoryStore(same tmp_path), verify get() returns the entry with correct content
   - `test_memory_store_get_missing_returns_none`: Verify store.get("nonexistent") returns None
   - `test_memory_store_remove`: Add entry, remove it, verify get() returns None, verify persistence (new store also returns None)
   - `test_memory_store_list_all`: Add 3 entries, verify list_all() returns all 3

3. Write async tests (plain `async def test_*()`, no decorator needed due to asyncio_mode=auto) for:
   - `test_retriever_hybrid_search`: Index 3 entries with distinct content, search for a term, verify results are non-empty and most relevant entry is ranked first
   - `test_retriever_bm25_only`: Set alpha=0.0, index entries, search — verify BM25 term-overlap ranking
   - `test_retriever_faiss_only`: Set alpha=1.0, verify FAISS-only ranking works
   - `test_retriever_format_context`: Call format_context() with results, verify it returns a non-empty string

All tests use `tmp_path` fixture for file isolation. All embeddings use `dtype=np.float32`. No network calls, no real LLM.
