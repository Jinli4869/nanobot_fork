<interfaces>
Locked task-layer shape:
```python
@dataclass(frozen=True)
class ShortcutRefNode:
    shortcut_id: str
    param_bindings: dict[str, str] = field(default_factory=dict)

@dataclass(frozen=True)
class BranchNode:
    condition: StateDescriptor
    then_steps: tuple["TaskNode", ...] = ()
    else_steps: tuple["TaskNode", ...] = ()

TaskNode = ShortcutRefNode | SkillStep | BranchNode
```

Task-skill contract target:
```python
@dataclass(frozen=True)
class TaskSkill:
    skill_id: str
    name: str
    description: str
    app: str
    platform: str
    steps: tuple[TaskNode, ...] = ()
    memory_context_id: str | None = None
    tags: tuple[str, ...] = ()
```

Serialization constraint:
```text
- serialized task nodes must use explicit discriminator values `shortcut_ref`, `atom_step`, and `branch`
- `atom_step` payload must wrap `SkillStep.to_dict()` output under a `step` key
- `BranchNode.then_steps` and `BranchNode.else_steps` must serialize recursively as lists of tagged nodes
```
