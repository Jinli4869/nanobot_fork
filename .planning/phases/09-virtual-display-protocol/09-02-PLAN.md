**Addition 1 — Error types (at module top, after imports):**
```python
class XvfbNotFoundError(RuntimeError):
    """Raised when the Xvfb binary is not installed on the system."""

class XvfbCrashedError(RuntimeError):
    """Raised when the Xvfb process exits unexpectedly before the X11 socket appears."""
```

**Addition 2 — Auto-increment with lock file pre-check:**
- Add constant `_MAX_RETRIES = 5`
- Refactor `start()` into an outer retry loop + inner `_try_start(display_num)` method
- The outer loop: for `attempt in range(_MAX_RETRIES)`, compute `num = self._display_num + attempt`, check `pathlib.Path(f"/tmp/.X{num}-lock").exists()` — if exists, log at DEBUG level `"Skipping display :%d — lock file exists (may be stale)"` and continue. Otherwise call `await self._try_start(num)`.
- If all retries exhausted, raise `RuntimeError(f"Could not acquire a free display after {_MAX_RETRIES} attempts")`

**Addition 3 — stderr=PIPE (not DEVNULL):**
- In `_try_start()`, change `stderr=asyncio.subprocess.DEVNULL` to `stderr=asyncio.subprocess.PIPE`
- Keep `stdout=asyncio.subprocess.DEVNULL`

**Addition 4 — Crash detection in socket poll loop:**
- In the socket polling loop inside `_try_start()`, after each `asyncio.sleep()`, check `self._process.returncode is not None`. If the process has exited before the socket appeared:
  - Drain stderr safely: `stderr_bytes = await self._process.stderr.read()` (safe because process has already exited)
  - Decode: `stderr_text = stderr_bytes.decode(errors="replace").strip()`
  - Set `self._process = None` (process is dead)
  - Raise `XvfbCrashedError(f"Xvfb exited with code {returncode} before socket appeared on :{num}. Stderr: {stderr_text}")`
- Wrap the `asyncio.create_subprocess_exec` call in a `try/except FileNotFoundError as exc:` that raises `XvfbNotFoundError("Xvfb is not installed. Install it with: apt install xvfb") from exc`

**Addition 5 — Drain stderr in stop():**
- In `stop()`, after `self._process.terminate()` and `await self._process.wait()`, drain stderr: `stderr_bytes = await self._process.stderr.read()` if `self._process.stderr` is not None. Log stderr content at DEBUG level if non-empty.

**Use `asyncio.wait_for()` for timeout enforcement** in `_try_start()` — wrap the socket polling loop in an async helper, and call `await asyncio.wait_for(self._poll_socket(socket_path), timeout=self._startup_timeout)`. On `asyncio.TimeoutError`, call `await self.stop()` then re-raise as `TimeoutError(f"Xvfb failed to create socket {socket_path} within {self._startup_timeout}s")`.

Keep `__all__` to export: `["XvfbDisplayManager", "XvfbNotFoundError", "XvfbCrashedError"]`
