    **Step 2: Create `opengui/backends/desktop.py`**

    Model directly on AdbBackend structure. Key implementation details:

    **Imports:** Import mss at module top level. Import pyautogui and pyperclip lazily inside methods (pyautogui's import-time DPI awareness call must happen after mss).

    **Class: LocalDesktopBackend**
    - `__init__(self)`: Set `self._screen_width = 0`, `self._screen_height = 0`. Set `pyautogui.PAUSE = 0.0` (lazy, on first use). Store `self._platform` by checking `platform.system()` once: `"Darwin"` -> `"macos"`, `"Linux"` -> `"linux"`, else `"windows"`.
    - `platform` property: return `self._platform`
    - `async def preflight(self)`: Try `pyautogui.position()`. On any exception, raise `RuntimeError("Enable Accessibility for Terminal/iTerm in System Settings > Privacy & Security > Accessibility")`.
    - `async def observe(self, screenshot_path: Path, timeout: float = 5.0)`:
      - `screenshot_path.parent.mkdir(parents=True, exist_ok=True)`
      - Capture with `mss.mss()` context manager, `sct.monitors[1]` for primary monitor
      - `logical_w, logical_h = monitor["width"], monitor["height"]`
      - `physical_w, physical_h = sct_img.size`
      - `img = Image.frombytes("RGB", (physical_w, physical_h), sct_img.bgra, "raw", "BGRX")`
      - If physical != logical: `img = img.resize((logical_w, logical_h), Image.LANCZOS)`
      - `img.save(str(screenshot_path), "PNG")`
      - Store logical dims in `self._screen_width`, `self._screen_height`
      - Call `fg_app = await self._query_foreground_app()`
      - Return `Observation(screenshot_path=str(screenshot_path), screen_width=logical_w, screen_height=logical_h, foreground_app=fg_app, platform=self._platform)`
    - `async def execute(self, action: Action, timeout: float = 5.0) -> str`:
      - Dispatch on `action.action_type`:
        - `"tap"`: `pyautogui.click(x, y)` where x,y from `_resolve_point(action)`
        - `"double_tap"`: `pyautogui.doubleClick(x, y)`
        - `"long_press"`: `pyautogui.rightClick(x, y)`
        - `"swipe"` / `"drag"`: `pyautogui.mouseDown(x1, y1, button="left")`, `pyautogui.moveTo(x2, y2, duration=(action.duration_ms or 300)/1000.0)`, `pyautogui.mouseUp(x2, y2, button="left")`
        - `"scroll"`: Move to point (center if no x/y), compute `clicks = max(1, abs(action.pixels or 120) // 120)`, direction from `action.text`: "down" -> `pyautogui.scroll(-clicks)`, "up" -> `pyautogui.scroll(clicks)`, "left" -> `pyautogui.hscroll(-clicks)`, "right" -> `pyautogui.hscroll(clicks)`
        - `"input_text"`: `pyperclip.copy(action.text or "")`, then `pyautogui.hotkey("command", "v")` on macOS, `pyautogui.hotkey("ctrl", "v")` on Linux/Windows
        - `"hotkey"`: `pyautogui.hotkey(*self._normalize_keys(action.key or []))`
        - `"wait"`: `await asyncio.sleep((action.duration_ms or 1000) / 1000.0)`
        - `"done"`: pass (no-op)
        - `"back"`: macOS `pyautogui.hotkey("command", "[")`, Linux/Windows `pyautogui.hotkey("alt", "left")`
        - `"home"`: macOS `pyautogui.hotkey("command", "shift", "h")` (Show Desktop), Linux `pyautogui.hotkey("super")`, Windows `pyautogui.hotkey("win", "d")`
        - `"open_app"`: macOS `await self._run_cmd("open", "-a", action.text)`, Linux `await self._run_cmd("xdg-open", action.text)`, Windows `await self._run_cmd("cmd", "/c", "start", "", action.text)`
        - `"close_app"`: macOS graceful `await self._run_cmd("osascript", "-e", f'quit app "{action.text}"')`, then force `await self._run_cmd("pkill", "-f", action.text)` on timeout. Linux `await self._run_cmd("pkill", "-f", action.text)`. Windows `await self._run_cmd("taskkill", "/IM", f"{action.text}.exe")`.
        - else: `raise ValueError(f"Unsupported action type for desktop backend: {action.action_type!r}")`
      - Return `describe_action(action)`

    **Helper methods:**
    - `_resolve_point(self, action: Action) -> tuple[int, int]`: Use `resolve_coordinate(action.x, self._screen_width, relative=action.relative)` and same for y/height. Return (x, y).
    - `_resolve_second_point(self, action: Action) -> tuple[int, int]`: Same using action.x2, action.y2.
    - `_normalize_keys(self, keys: list[str]) -> list[str]`: Apply platform-aware modifier map. Map dict: `{"darwin": {"cmd": "command", "option": "alt", "super": "command"}, "linux": {"cmd": "win", "super": "win", "option": "alt"}, "win32": {"cmd": "win", "option": "alt", "super": "win"}}`. Use `sys.platform` to select map. Return `[map.get(k.lower(), k.lower()) for k in keys]`.
    - `async def _run_cmd(self, *args: str, timeout: float = 5.0) -> str`: Use `asyncio.create_subprocess_exec(*args, stdout=PIPE, stderr=PIPE)`, `asyncio.wait_for(proc.communicate(), timeout)`, handle TimeoutError by killing process and returning "". Return decoded stdout.
    - `async def _query_foreground_app(self) -> str`: macOS: `osascript -e 'tell application "System Events" to get name of first process where it is frontmost'`. Linux: `xdotool getactivewindow getwindowpid` + `/proc/{pid}/comm`. Windows: stub returning "unknown". Return "unknown" on any error.
