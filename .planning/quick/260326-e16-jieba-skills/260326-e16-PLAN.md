<objective>
Integrate jieba word segmentation into the BM25 tokenizer so that Chinese queries produce word-level tokens instead of character-level tokens, significantly improving BM25 precision for Chinese text in both memory retrieval and skills search.

Output: Updated `_tokenize()` with jieba integration, optional dependency declaration, and comprehensive Chinese BM25 tests.

```python
def _tokenize(text: str) -> list[str]:
    """Tokenize *text* with CJK char-level splitting + Latin word splitting."""
    text = text.lower()
    tokens: list[str] = []
    buf: list[str] = []
    for ch in text:
        if _is_cjk(ch):
            if buf:
                tokens.append("".join(buf))
                buf.clear()
            tokens.append(ch)  # <-- This is the problem: each CJK char = 1 token
        elif re.match(r"\w", ch):
            buf.append(ch)
        else:
            if buf:
                tokens.append("".join(buf))
                buf.clear()
    if buf:
        tokens.append("".join(buf))
    return tokens
```
