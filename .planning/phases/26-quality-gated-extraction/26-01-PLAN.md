Producer implementation details:
- Use `_PARAM_RE = re.compile(r"\{\{(\w+)\}\}")` to scan step targets for `{{param_name}}` placeholders
- Infer `ParameterSlot(name=param_name, type="string", description=f"Value for {param_name}")` for each unique placeholder
- Convert each step dict to `SkillStep(action_type=step["action"]["action_type"], target=step.get("model_output", ""), ...)` — map from trajectory step event shape to SkillStep
- Map `valid_state` from step dict observation or model output hints to `StateDescriptor(kind="screen_state", value=...)` as preconditions — skip entries where value is empty string or case-insensitive "no need to verify"
- Map `expected_state` similarly to postconditions
- Normalize app identifier via `normalize_app_identifier(platform, app)`
- Generate `skill_id` via `str(uuid.uuid4())`
- Set `name` from first step's action type + target slug (snake_case, max 50 chars)
- Set `description` from metadata task description if available in step context, else "Extracted from trajectory"
- Set `created_at` via `time.time()`
