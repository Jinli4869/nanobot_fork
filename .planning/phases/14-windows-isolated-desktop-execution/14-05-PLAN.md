   - `shutdown()` must send the `shutdown` command first, wait up to 1 second, terminate/kill on timeout, close process pipes, unlink the control path, then call `await self._display_manager.stop()`, and only then release the runtime lease
   - keep the ready/cleanup log lines and include `cleanup_reason=`, `display_id=`, and `backend_name=windows_isolated_desktop`
