# Phase quick Plan 260326-e16: jieba Chinese Tokenization in BM25 Summary

**One-liner:** jieba word segmentation integrated into `_tokenize()` via optional import guard with char-level fallback, so "浏览器" is one BM25 token instead of three characters.

## Objective

Integrate jieba word segmentation into the BM25 tokenizer so Chinese queries produce word-level tokens instead of character-level tokens, improving BM25 precision for Chinese text in both memory retrieval and skills search.

## What Was Built

### `opengui/memory/retrieval.py`

Added jieba import guard at module level:
```python
try:
    import jieba as _jieba
    _JIEBA_AVAILABLE = True
except ImportError:
    _jieba = None
    _JIEBA_AVAILABLE = False
```

Refactored `_tokenize()` into three functions:
- `_tokenize_charlevel(text)` — existing char-level CJK splitting (fallback)
- `_tokenize_jieba(text)` — jieba.lcut() with punctuation/whitespace filter
- `_tokenize(text)` — dispatcher: jieba if available, else char-level

### `pyproject.toml`

- Added `cjk = ["jieba>=0.42.1"]` optional dependency group
- Added `"jieba>=0.42.1"` to `dev` group for test runs

### `tests/test_opengui_p1_memory.py`

Added 6 new tests covering:
1. `test_tokenize_chinese_word_segmentation` — "浏览器" as single token
2. `test_tokenize_mixed_latin_cjk` — "open浏览器test" correctly split
3. `test_tokenize_english_unchanged` — "hello world" regression guard
4. `test_tokenize_fallback_char_level` — monkeypatches _JIEBA_AVAILABLE=False
5. `test_retriever_bm25_chinese_search` — BM25-only alpha=0.0 Chinese query
6. `test_skill_library_search_chinese_bm25` — SkillLibrary Chinese BM25 search

## Verification Results

All 4 plan success criteria confirmed:

1. `_tokenize("打开浏览器搜索内容")` → `['打开', '浏览器', '搜索', '内容']` (word-level)
2. `_tokenize("hello world test")` → `['hello', 'world', 'test']` (unchanged)
3. SkillLibrary.search() inherits improvement automatically (no code changes needed)
4. When `_JIEBA_AVAILABLE=False`, `_tokenize("浏览器")` → `['浏', '览', '器']` (fallback)

Test counts: 19 memory tests pass, 22 skills tests pass.
