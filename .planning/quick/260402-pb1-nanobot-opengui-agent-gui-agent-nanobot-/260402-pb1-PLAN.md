<objective>

main agent on `qwen3.5-plus` while selecting a different GUI model/provider, then add an
opt-in GUI-task evaluation hook exposed through `~/.nanobot/config.json`.

The evaluation path should reuse the same judge logic as `eval/eval.py`, but in a
single-trajectory form suitable for one GUI run at a time.
</objective>
