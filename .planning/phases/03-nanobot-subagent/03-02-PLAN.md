`__init__(self, gui_config, provider, model, workspace)`:

  - `"adb"`: `from opengui.backends.adb import ADBBackend; ADBBackend(serial=gui_config.adb.serial)`
  - `"local"`: `from opengui.backends.local import LocalDesktopBackend; LocalDesktopBackend()` (Phase 4 — raise NotImplementedError if not yet available)

- Store backend as `self._backend`
- Compute skill library path: `workspace / "gui_skills" / self._backend.platform`
- Create `SkillLibrary(store_dir=skill_lib_path, merge_llm=self._llm_adapter)` and store as `self._skill_library`

Properties:
- `name` -> `"gui_task"`
- `description` -> `"Execute a GUI automation task on a device. The task is performed by a vision-action agent that observes screenshots and executes actions. Returns a structured result with success status, summary, and trace path."`
- `parameters` -> JSON Schema with `task` (string, required) and `backend` (string, optional, enum ["adb", "local", "dry-run"])

`async def execute(self, task: str, backend: str | None = None, **kwargs) -> str`:
1. Create timestamped run directory: `run_dir = self._workspace / self._gui_config.artifacts_dir / datetime.now().strftime("%Y-%m-%d_%H%M%S")`; `run_dir.mkdir(parents=True, exist_ok=True)`
2. Select backend: if `backend` override is provided and differs from config, construct a new backend (same logic as __init__); otherwise use `self._backend`
3. Create fresh `TrajectoryRecorder(output_dir=run_dir, task=task, platform=active_backend.platform)`
4. Create `GuiAgent(llm=self._llm_adapter, backend=active_backend, trajectory_recorder=recorder, model=self._model, artifacts_root=run_dir, max_steps=self._gui_config.max_steps, skill_library=self._skill_library, skill_threshold=self._gui_config.skill_threshold)`
5. `result = await agent.run(task=task)`
6. **Auto skill extraction** (wrapped in try/except to never break the tool result):
   - Find trace file: `trace_path = run_dir / "trace.jsonl"`
   - If trace_path exists:
     - `from opengui.skills.extractor import SkillExtractor`
     - `extractor = SkillExtractor(llm=self._llm_adapter)`
     - `skill = await extractor.extract_from_file(trace_path, is_success=result.success)`
     - If `skill is not None`: `await self._skill_library.add_or_merge(skill)`; log info
     - If `skill is None`: log debug "No skill extracted"
   - Catch `Exception` broadly — log warning but do not propagate
7. Return `json.dumps({"success": result.success, "summary": result.summary, "trace_path": str(result.trace_path) if result.trace_path else str(trace_path), "steps_taken": result.steps_taken, "error": result.error})`
